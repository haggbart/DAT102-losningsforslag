# DAT102-losningsforslag
dat102 løsningsforslag

NB. innholdet er ikke ment som fasit, men er laget for å diskutere oppgaver og løsninger i dat102 med studenter på HVL i førde.
Løsningene/koden er i flere tilfeller mangelfull(e) og/eller ineffektiv(e) som følge av latskap, at oppgaver oppfattes kjedelige eller begrensninger slik oppgavene er definert.
